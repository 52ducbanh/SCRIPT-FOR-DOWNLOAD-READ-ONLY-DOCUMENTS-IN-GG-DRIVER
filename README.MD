# SCRIPT FOR DOWNLOAD READ-ONLY DOCUMENTS IN GG DRIVER

## 1. DESCRIPTION

Some people face the problem of not being able to download documents on Google Drive because the authors have restricted downloadable permissions. This script will let you download it easily :>

## 2. GETTING STARTED

### NOTES BEFORE RUNNING THIS SCRIPT

1. You need to scroll all the pages in documents to load full pages that need to download.
2. The more you zoom in before running the script, the higher the resolution and sharper the output will be.

### HOW TO RUN SCRIPT

1. Press simultaneously Ctrl + Shift + J (Windows / Linux) or Cmd + Option + J (MacOS) to open Console.
2. If your browser restricts you from pasting script, remember to type `allow pasting` into the Console first, then paste the script before giving up hehe :)
3. Paste the script into the Console tab and press Enter to launch it.
4. Wait until the script reaches 100% - it will automatically download the file for you.
